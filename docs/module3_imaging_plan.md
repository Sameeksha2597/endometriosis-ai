# GLENDA / ENID Imaging Module Plan

## Objective

Develop a computer vision model capable of classifying laparoscopic images as:

- Endometriosis
- No Endometriosis

## Dataset

- Total Images: 1022
- Endometriosis: 533
- No Endometriosis: 489

## Planned Pipeline

1. Data Loading
2. Train/Validation/Test Split
3. Image Augmentation
4. Transfer Learning (ResNet50)
5. Model Training
6. Evaluation
7. Confusion Matrix
8. ROC-AUC Analysis
9. Grad-CAM Explainability

## Success Metrics

- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC
